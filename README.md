# IDASignsrch
IDAPython Plugin for searching signatures, use xml signature database from [IDA_Signsrch](https://github.com/nihilus/IDA_Signsrch)

![https://i.imgur.com/kBj8KW1.png](https://i.imgur.com/kBj8KW1.png)

---

## Installation

### Method 1: Using hcli (Recommended)

```
hcli plugin install IDASignsrch
```

### Method 2: Manual Installation

1. Download or clone this repository.
2. Copy `signsrch.py` and `signsrch.xml` into your IDA plugins directory:
   - **Windows:**
     `%APPDATA%\Hex-Rays\IDA Pro\plugins\`
   - **macOS/Linux:**
     `~/.idapro/plugins/`
3. Restart IDA.

---

## How to use
Edit -> Plugins -> Signsrch
